# Arduino LM35 Temperature Monitor

## Mo ta
Du an doc nhiet do tu cam bien LM35 bang Arduino va gui du lieu qua Serial.

## Tinh nang
- Doc nhiet do tu cam bien LM35
- Ho tro nhieu kenh cam bien
- Gui du lieu qua Serial
- Co the mo rong de hien thi tren ung dung PC

## Phan cung can thiet
| Linh kien | So luong | Ghi chu |
|-----------|----------|---------|
| Arduino UNO | 1 | Bo dieu khien chinh |
| Cam bien LM35 | 2-3 | Doc nhiet do |
| Day noi | Vua du | Ket noi cac linh kien |

## Huong dan su dung
1. Ket noi cam bien LM35 vao Arduino.
2. Nap chuong trinh vao board Arduino.
3. Mo Serial Monitor.
4. Quan sat du lieu nhiet do duoc gui ra.
## Cau truc thu muc
- firmware/: Chua chuong trinh Arduino
- pc_app/: Chua ung dung may tinh
- docs/: Chua tai lieu
- libs/: Chua thu vien
- simulation/: Chua file mo phong
